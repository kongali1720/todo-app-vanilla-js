## Cara Pakai

1. Clone atau download repo ini  
   ```bash
   git clone https://github.com/kongali1720/todo-app-vanilla-js.git
   ```

---

Buka file index.html di browser favoritmu
Mulai tambahkan tugas di kolom input dan klik tombol Tambah
Klik tugas untuk menandainya selesai
Klik tombol Hapus di samping tugas untuk menghapusnya

HTML5
CSS3
